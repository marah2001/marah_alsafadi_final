## ✨ عن المشروع (About)
تطبيق أندرويد متكامل لمتجر مستحضرات تجميل، يتميز بتصميم عصري وأداء سلس. تم بناء التطبيق باستخدام **Jetpack Compose** ويوفر تجربة تسوق كاملة من تصفح المنتجات وحتى إضافتها للسلة والمفضلة.

---

## 🚀 المميزات (Features)
* **🏠 Home Screen:** عرض جذاب للمنتجات مع بانر "Shop Now".
* **🛒 Cart System:** إضافة المنتجات للسلة مع تحديث فوري للكمية (Badge) والسعر الإجمالي.
* **❤️ Favorites:** نظام مفضلة ذكي يتيح للمستخدم حفظ منتجاته المفضلة.
* **📱 Product Details:** صفحة تفصيلية لكل منتج تعرض الصور والوصف والسعر.
* **👤 Account Page:** إدارة الحساب الشخصي، طرق الدفع، وعناوين الشحن.
* **🗺️ Google Maps:** ميزة تحديد موقع الشحن في غزة.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **Language:** Kotlin 100%
* **UI Framework:** Jetpack Compose
* **Navigation:** Compose Navigation
* **Architecture:** Clean Code principles
